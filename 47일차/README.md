# SpendScope 💸

> **사회초년생을 위한 심플하고 강력한 지출 관리 앱**

SpendScope는 복잡한 수입 관리 대신 오직 '지출'에만 집중하여, 사용자의 소비 패턴을 직관적으로 분석해주는 모바일 최적화 웹 애플리케이션입니다.

## ✨ 주요 기능

- **실시간 지출 대시보드**: 이번 달 총 지출 및 식비 지표를 직관적으로 확인.
- **카테고리별 스마트 분석**: 파이 차트와 바 차트를 통해 소비 비중을 시각적으로 분석.
- **날짜별 지출 그룹화**: 언제 어디서 썼는지 한눈에 파악하는 타임라인 리스트.
- **모바일 최적화 UI**: 넉넉한 터치 영역과 18px 본문 폰트로 모바일 사용성 극대화.
- **다이내믹 인터랙션**: 차트 클릭 시 상세 내역 강조 등 풍부한 사용자 경험 제공.

## 🚀 시작하기

이 프로젝트는 별도의 서버 설치 없이 브라우저에서 바로 확인할 수 있는 통합 데모를 제공합니다.

1. 이 저장소를 클론하거나 다운로드합니다.
2. 폴더 내의 `index.html` 파일을 크롬 웹 브라우저로 엽니다.
3. 하단 네비게이션 바를 이용해 각 화면을 탐색해 보세요.

## 🛠 기술 스택

- **React**: UI 컴포넌트 기반 아키텍처
- **Context API**: 전역 상태 관리
- **BEM CSS**: 확장 가능하고 유지보수가 쉬운 스타일링 구조
- **Semantic HTML**: 논리적인 문서 구조 설계

## 📱 디자인 시스템

- **Primary Color**: `#2563EB` (Primary Blue)
- **Typography**:
  - Display: 28px (Main Amount)
  - Heading: 20px (Section Title)
  - Body: 18px (Content)
  - Caption: 14px (Secondary Info)
- **Grid System**: 8px Base Grid
